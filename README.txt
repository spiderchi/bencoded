-> Library is developed with Javascript using ES6 feature and used Module defination.

To run script need to have node installed on computer
After installing the node, steps to run script

1) Extract the provided zip file
2) Place all code files under pixmob folder
3) using command prompt or terminal navigate to pixmob folder
4) Run "node encode.js" --> this will console out the encoded data
5) Run "node decode.js" --> this will console out the decoded data from provided encoded data
